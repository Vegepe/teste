Aula 2 - Domínio e hospedagem

  • Situação 1: HTML e CSS no servidor;
  • Situação 2: Guardado no meu pc;
  • Situação 3: Guardado em outro pc;

URL:

  • é a ponte;
  • precisa de um domínio → identifica o site;
  • é hospedado por um servidor → onde o URL fica armazenado.

Domínio:

  • é um nome único;
  • é pago anualmente;
  • tem vários TLDS (Top Level Domain [.com]).

Hospedagem:

  • é o espaço para armazenar arquivos
  • é paga mensalmente
  • lida com espaço, memória e recursos.

  • Uniform Resource Locator (URL) → TUDO;
  • Generic TLD: instituições (.gov);
  • Country Code TLD: código de país (.br).

Aula 1 - Diferença entre HTML, CSS e JavaScript

  • HTML → Hypertext Markup Language → Focada em conteúdo!!! (textos, imagens, vídeos, tabelas);
  • CSS → Cascading Style Sheets → Focada em design!!! (cores, sombras, tamanho, posicionamento);
  • JS → JavaScript → Trabalha com interatividade!!! (menus, animações, popups, validações).

HTML usa tags (marcações):

ex: <barbecue>BOMDIA</barbecue>


CSS usa seletores contendo declarações com propriedades e seus valores:

ex: h1 {
        font-family: arial;
    }
