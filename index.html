Arpanet criada pra ser um "backup" entre os centros na guerra

Protocolos são idiomas únicos, universais, que todo mundo consegue falar. NCP foi o primeiro.

IP → Internetwork Protocol

TCP/IP → Usado até hoje, criado por Bob Khan e Vint Cerf.

Arpanet se separou em 3 redes:

  • Milnet → Militar
  • Nsfnet → Universidades
  • Muitas redes comerciais
Obs: nsf = National Science Foundation

Queriam todos se comunicarem-ser-lhe-eis, daí veio o Internetworking

https://www.submarinecablemap.com/ → Mapa dos fios subterrâneos de fibra ótica.

IP → Identifica os pontos de envio e recebimento de dados, como se fosse o endereço de uma casa.
